Hi there, I'm Krisztian

I am a Hungarian web designer and I create custom web designs. I'm skilled at writing well-designed, testable and efficient code using current best practices in Web development. I'm a fast learner, hard worker and team player who is proficient in making creative and innovative web pages.

My Skills

  • Responsive Web Design

    I also cover the concept of mobile-first design. Through my work, I focus on designing an experience that works across different screen sizes.

  • UX Auditing
